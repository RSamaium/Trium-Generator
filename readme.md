# Trium Generator

v0.1.0

# Install Trium

    npm install -g trium
    
# Create Trium environment

    trium install
    
# Create a bundle

    trium create <Bundle Name>
    
# Generate bundle files

    trium generate <Bundle Name>
    
Options :
* -s : Only server files
* -c : Only client files
    
# Update bundle files

    trium update <Bundle Name>
    
# Delete bundle files

    trium uninstall <Bundle Name>
    
Options :
* -s : Only server files
* -c : Only client files
